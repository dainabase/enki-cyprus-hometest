# Inspirations de référence — ENKI Realty Design System v2

> Références visuelles à mobiliser pour la direction **Premium européen poussée loin**

## Immobilier premium / luxe

### Sotheby's International Realty
- URL : https://www.sothebysrealty.com
- Ce qu'on garde : espace blanc généreux, hiérarchie typographique forte, photographie en grand format sans texte intrusif, palette très neutre (blanc, noir, gris), CTA discrets mais bien placés
- Ce qu'on rejette : ton trop intimidant, zéro chaleur, absence de data visualisations

### Knight Frank
- URL : https://www.knightfrank.com
- Ce qu'on garde : autorité data (Wealth Report), gris charcoal comme couleur institutionnelle, tableaux de données élégants, navigation dense mais lisible
- Ce qu'on rejette : trop corporate, manque de modernité interactive

### Christie's International Real Estate
- URL : https://www.christiesrealestate.com
- Ce qu'on garde : typographie éditoriale premium (serif display possible), storytelling par propriété, mise en page de magazine
- Ce qu'on rejette : trop ancien-riche, pas assez inclusif pour clientèle entrepreneurs 30-45 ans

### Aman Resorts (référence hospitality luxe)
- URL : https://www.aman.com
- Ce qu'on garde : sobriété extrême, photographie immersive, spacing généreux, sensation d'exclusivité sans ostentation
- À adapter : on reste plus chaleureux et accessible

## Fintech et produits sophistiqués

### Stripe
- URL : https://stripe.com
- Ce qu'on garde : typographie impeccable, animations signature (backgrounds animés), data visualisations claires, code samples inline
- Ce qu'on cherche à répliquer : le "wow" technique sans jamais sacrifier la lisibilité

### Linear
- URL : https://linear.app
- Ce qu'on garde : palette monochrome sophistiquée, micro-interactions parfaites, typography premium Inter
- Applicable à ENKI : oui pour le chat et l'admin

### Vercel
- URL : https://vercel.com
- Ce qu'on garde : noir profond + accent vibrant, contraste maximal, typography display forte
- Pas applicable : trop tech pour investisseurs 50+

## Hospitality premium accessible

### Airbnb Plus
- URL : https://www.airbnb.com/plus
- Ce qu'on garde : raffinement accessible, photographie premium sans intimidation, CTA chaleureux
- Exactement le registre émotionnel qu'on vise

## Rédactionnel éditorial

### The New York Times (real estate section)
- URL : https://www.nytimes.com/section/realestate
- Ce qu'on garde : serif éditorial pour les titres (typo évocatrice d'autorité journalistique)
- Question ouverte : Fraunces ? Playfair Display ? Canela ? GT Super ?

### Monocle Magazine
- URL : https://monocle.com
- Ce qu'on garde : layout éditorial magazine, palette warm/neutrals, confiance institutionnelle
- Très proche du ton ENKI visé

## Directions typographiques à tester

### Direction 1 — Inter-only (minimaliste moderne)
- Hero : Inter ExtraLight 8xl tracking-tighter
- Sub : Inter Regular xl
- Body : Inter Regular base
- Raisonnement : pureté, tech-forward, accessible, déjà maîtrisé

### Direction 2 — Serif display + Inter (éditorial premium)
- Hero : **Fraunces** Regular/Light 7xl tracking-tight
- Sub : Inter Regular xl
- Body : Inter Regular base
- Raisonnement : autorité éditoriale, sensation de publication premium, différenciation vs concurrence tech

### Direction 3 — Grotesque display + Inter (modernisme contemporain)
- Hero : **GT Super Display** ou **Canela** 7xl
- Sub : Inter Regular xl
- Body : Inter Regular base
- Raisonnement : luxe contemporain, plus distant, moins chaud

## Directions de palette à tester

### Palette A — Cyprus Mediterranean évoluée
- Garder ocean blue `#57B9D6` mais raffiner vers une version plus profonde
- Ajouter un navy foncé pour les h1/h2 (texte de marque)
- Neutrals warm (beige, sable)
- Accent terra cotta conservé

### Palette B — Premium European Navy
- Navy profond `hsl(220 30% 15%)` en primary
- Or champagne `hsl(40 60% 70%)` en accent premium
- Neutrals chauds (crème, beige clair)
- Gris charcoal pour data

### Palette C — Editorial Luxury Warm
- Beige/sable neutre dominant
- Terra cotta `hsl(15 50% 50%)` en accent
- Or métallique `hsl(45 70% 55%)` pour Golden Visa
- Noir profond pour typographie
- Très peu de bleu

## Motion et interactions

### Signatures à explorer
- **Parallax éditorial** : images se déplacent lentement au scroll (Sotheby's)
- **Stagger reveals** : éléments apparaissent en cascade (Linear)
- **Hover raffinés** : scale 1.02 max + shadow-premium (pas 1.05 qui est trop)
- **Cursor custom** sur les property cards (Christie's)

### À éviter
- Emojis animés (règle stricte)
- Effets "wow gratuit" qui ne servent pas la conversion
- Animations de chargement ludiques (on est premium, pas startup)

## Chiffres et mots-clés pour Claude Design

Quand on prompte Claude Design, utiliser ces mots pour ajuster le ton :
- "plus éditorial", "plus institutionnel", "plus magazine"
- "moins startup-y", "moins tech-bro", "moins cheap"
- "densité contrôlée", "hiérarchie forte", "espace blanc généreux"
- "confiance datarique", "autorité calme"
- "warm neutrals", "tokens métalliques"
