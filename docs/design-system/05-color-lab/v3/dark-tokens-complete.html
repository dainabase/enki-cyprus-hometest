Couche 5 Dark Mode · 7 tokens dérivés du fond D2

Le système dark complet.

Sur la base du fond D2 #0F1A2A validé, j'ai dérivé les 6 autres tokens en respectant la cohérence chromatique du light et les seuils WCAG AA.

Card est un cran plus clair que le bg (élévation visible). Foreground reprend exactement le papier bible du light #FCFAF5 — système d'inversion symétrique qui simplifie tout. Foreground muted est calibré pour rester lisible sans rivaliser avec le foreground principal. Accent est l'Aero éclairci (#57B9D6 au lieu de #3D8FB0) parce que sur fond sombre, l'Aero méditerranéen passe en dessous du seuil de contraste. Soft est l'Aero-900, une version très foncée pour les bulles user et bordures. Warm est la Chamoisée éclaircie pour rester lisible.

Couche 5 entièrement fermée si tu valides.

Le système dark complet est posé. Symétrie chromatique parfaite avec le light (mêmes hue, luminosités inversées). Tous les contrastes WCAG tiennent (5 paires AAA, 1 paire AA pour Warm qui reste largement au-dessus du seuil). Identité ENKI préservée (papier bible reste papier bible, Aero reste Aero, Chamoisée reste Chamoisée).

Si tu valides, je verrouille la Couche 5 entièrement aujourd'hui : 7 tokens light + 7 tokens dark = 14 tokens définitifs, prêts à être implémentés dans le code et documentés sur Notion + GitHub.
